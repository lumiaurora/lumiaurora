## About Me

I am Lumi Aurora, also known as Yuki: an 18-year-old builder from Finland who loves computer science, game systems, cybersecurity, and making technical ideas feel alive.

I work across Python, C, and Java, and I am especially drawn to projects that mix creativity with precision: Minecraft plugins and mods, low-level systems, reverse engineering, binary exploitation, cryptography, web hacking, machine learning experiments, networking, infrastructure, and math-heavy problem solving.

If it is technical, a little ambitious, and fun to figure out, I probably want to build it.

## What I Build

- Minecraft plugins, mods, and custom mechanics that make worlds feel more alive
- Python tools, automation, and machine learning experiments that turn ideas into something real
- C projects that help me understand systems, memory, performance, and what happens under the hood
- Java projects for structured engineering, larger codebases, and Minecraft development
- Cybersecurity work across reverse engineering, binary exploitation, cryptography, and web hacking
- Networking, firewall configuration, infrastructure, and the systems that keep software moving
- Math-driven solutions where logic, patterns, and clean thinking matter
- Software development in general: from ideas and experiments to tools people can actually use
